## 🚀 What It Does

JobHive aggregates live job postings from multiple job boards into one clean, searchable interface. No switching between tabs — search once and see results from everywhere.

**Features:**
- Live job search across LinkedIn, Indeed, Glassdoor, ZipRecruiter & Monster
- Filter by Full-time, Part-time, Contract, Remote
- Sort by newest, salary high to low, salary low to high
- Save favourite jobs
- Click any job → full details → apply directly on the original job board
- Works on mobile and desktop
